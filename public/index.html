Store api

This API is built on top of Store apiAPI. It enables developers to access data programmatically using a REST API. This documentation describes how to request data from the API and how to interpret the response.

Search

The search functionality allows users to query the product database with various parameters to find relevant items. This endpoint supports filtering, sorting, and pagination.

GET

https://___________/api/v1/products/?query=...

Common query parameters:
Parameter Description Type
query= full-text query string
numericFilters=

filter on a specific numerical condition (< , <=, =, >or >=).

string
page= page number Integer

Examples

{{URL}} shortcut for https://________/api/v1/products

Sort

The sort parameter allows you to specify the field by which the results should be sorted.
You can also include a hyphen (-) before the field name to indicate descending order.
The results will be sorted in ascending order by default.

GET

{{URL}}?sort=name, -price

{{URL}}?sort=name, -price
{
 "AllProducts":[
  {
   "featured":false,
   "rating":4.3,
